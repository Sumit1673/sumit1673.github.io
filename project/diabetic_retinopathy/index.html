Diabetic Retinopathy Analysis

Oct 2020
Code DL Model Code Video Dataset

Description:

  1. Chronic or poorly controlled high blood sugar can result in damage to the blood vessels in the retina. Diabetic retinopathy occurs when the damaged blood vessels leak blood and other fluids into your retina, causing swelling and blurry vision.
  2. The web app utilizes the power of cloud to reach to the hands of doctors.
  3. The doctors can create their account and can add patients with their retina images.
  4. Deep learning based classification upto severity level 4 (0 being No DR) gives result in less than 2 mins.

  5. The application utilizes the power of DL to classify images. Used ResNet-101 for this task. Model is trained on 9GB of retina images on GPU with 16GB memory.

  6. PyTorch framework is used for dataset management, training and testing.

  7. Django is used for web development. MongoDB is used as database. OneToOne and ManyToOne data model is used for database management.
    AWS platform is used as our cloud platform for launching the web app.
